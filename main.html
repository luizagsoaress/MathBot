Sobre
Esta página web foi desenvolvida como um projeto de estudo sobre HTML5, CSS e JavaScript. O banco de dados utilizado para a autenticação dos usuários e para o armazenamento de informações foi o Firebase. As chaves privadas, assim como o código PROLOG para a geração de respostas, foram hospedados em um servidor Node.js com Express, que está em execução no Render.
